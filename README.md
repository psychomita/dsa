## Run Locally

### Clone the project

```bash
git clone https://github.com/psychomita/cp
```

### Go to the project directory

```bash
cd cp
```

### Go to the specific folder

```bash
cd <Folder Name> ...
```

### Compile & Run the program

- Compile the program

    ```bash
    g++ <filename>.c -o <filename>
    ```

- Run the program

    ```bash
    ./<filename>
    ```
